# Easyfix_CRM_UI

Internal CRM frontend for the EasyFix platform. Next.js 15 App Router, consumes `EasyFix_Backend` at `http://localhost:5100/api`.

## Quick start

```bash
cp .env.example .env.local
npm install
npm run dev
```

Opens at http://localhost:5180.

See `CLAUDE.md` for architecture notes.
